Использование в HTML

Для использования слайдера, в HTML необходимо создать неупорядоченный список (ul), обернуть его в элемент "div" с классом "carousel", в каждый элемент списка поместить "div" с классом "carousel-item", который и является контейнером слайда. Ширина слайда задаётся в настройках слайдера. По умолчанию - 300px. Слайд может содержать изображения и текстовые блоки. Пример подключения в HTML:

Настройка плагина

  • horisontal - параметр, определяющий ориентацию слайдера; значение по-умолчанию: горизонтальная;
  • slideWidth - параметр, определяющий ширину слада в пикселях; значение по-умолчанию: 300px;
  • cyclic - параметр, определяющий будет ли прокрутка циклической; значение по-умолчанию: true (прокрутка будет циклической);
  • autoscrolling - параметр, определяющий будет ли циклическая прокрутка выполнятся автоматически; значение по-умолчанию: false (не будет);
  • autoscrollingSpeed - параметр, определяющий интервал в миллисекундах между шагами автоматической прокрутки; значение по-умолчанию: 5000;
  • slideDuration - параметр, определяющий длительность одного перемещения в миллисекундах; значение по-умолчанию: 500;
  • shiftIncrement - параметр, определяющий количество слайдов, перемещаемых за один шаг; значение по-умолчанию: 2;
  • imagesPerFrame - параметр, определяющий количество слайдов на экране; используется только при вертикальной прокрутке, при горизонтальной прокрутке количество слайдов на экране подбирается скриптом исходя из ширины слайда и доступного пространства по горизонтали таким образом, чтобы количество отображаемых слайдов было максимальным; значение по-умолчанию: 3;
